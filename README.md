#Roomverse
Roomverse is a smart roommate-matching platform designed for students and young professionals in India. It helps users find compatible living partners by analyzing behavioral traits, lifestyle choices, and personal preferences, ensuring a harmonious co-living experience.

🚀 Features
Smart Matching: Advanced filtering based on habits, budgets, and daily schedules.

User Profiles: Detailed descriptions highlighting lifestyle preferences and behavioral traits.

Responsive UI: A clean, modern interface designed for seamless navigation across all devices.

🛠️ Tech Stack
Frontend: React.js, Tailwind CSS

Backend: Node.js, Express.js

Database: MongoDB
